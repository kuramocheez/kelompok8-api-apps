CARA MENGGUNAKAN:
File RAR
1. Copy dan Extract file RAR pada direktori xampp/htdoc.
2. Nyalakan Apache dan MySQL pada XAMPP.
3. Buat database dengan nama kelompok8_app pada MySQL, kemudian import database yang terdapat pada folder db.
4. buka tab baru pada web browser, ketikkan localhost/kelompok8.

Github
1. Clone dan simpan pada direktori xampp/htdoc.
2. Nyalakan Apache dan MySQL pada XAMPP.
3. Buat database dengan nama kelompok8_app pada MySQL, kemudian import database yang terdapat pada folder db.
4. buka tab baru pada web browser, ketikkan localhost/kelompok8.
